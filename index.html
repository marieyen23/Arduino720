Code Explanation

Libraries

  1. DHT: Used for reading temperature and humidity from a DHT11 sensor.
  2. Ultrasonic: Used for interfacing with an ultrasonic ranger to measure distance.
  3. TFT_eSPI: Library for interfacing with a TFT display.
  4. Wire: I2C communication library.
  5. Digital_Light_TSL2561: Used for reading light intensity from a digital light sensor.
  6. Grove_LED_Bar: Library for controlling Grove LED bars.
  7. SoftwareSerial: Allows serial communication on other digital pins.

Pin Definitions

  • D0 (Motion Sensor): Detects motion.
  • D2 (Circular LED): Circular LED for visual alerts.
  • D4 (Ultrasound Ranger): Ultrasonic sensor for measuring distance.
  • D6 (Temperature and Humidity Sensor): DHT11 sensor for temperature and humidity.
  • I2C Port (Digital Light Sensor): I2C port for the digital light sensor.
  • WIO_KEY_B: Button to activate temperature and humidity sensor.
  • WIO_KEY_C: Button to activate motion sensor.
  • WIO_5S_PRESS: Button to activate low beam and high beam.

Constants and Variables

  • speakerPin: Pin for the buzzer.
  • DHTTYPE and DHTPIN: Type and pin for the DHT sensor.
  • MOTION_SENSOR: Pin for the motion sensor.
  • frequency: Frequency for the buzzer.
  • light: Variable to store the light intensity.
  • x: Boolean variable used in the main loop.

Setup Function

Initializes various sensors and peripherals. Sets up the TFT display and LED bar.

Loop Function

Checks for a serial command to start the system. Calls getReadyMessage and startEngine functions in response to the command.
